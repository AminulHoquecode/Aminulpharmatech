# Pharmatech

A Python library.

## Installation

```bash
pip install pharmatech
```

## Usage

```python
import pharmatech
```

## Development

To set up the development environment:

1. Clone the repository
2. Create a virtual environment:
   ```bash
   python -m venv venv
   .\venv\Scripts\activate  # On Windows
   ```
3. Install development dependencies:
   ```bash
   pip install -e ".[dev]"
   ```

## Testing

To run tests:

```bash
pytest
```
